  1. Grid Layout
    1. A two-dimensional grid-based layout system that was established mainly to handle the layout issues that CSS coders have been experiencing for years.
    2. A container element that uses the display: grid property is the first step in using grid layout.
    3. The grid's layout is primarily left to right and top to bottom
  2. Document Object Model (DOM)
    1. An interface that is cross-platform and language-independent and treats XML/HTML documents as a tree structure with each node acting as an object to represent a different parts of the document.
    2. Represents a document that contains a logical tree.
    3. The node at the end of each branch of the tree is filled with objects.
  3. Platform as a Service (PaaS)
    1. PaaS providers handle every aspect of provisioning hardware, network bandwidth, and other services for us.
    2. Github provides a service called "Github Pages" that makes it very simply to publish your own web page.
